Technical Experience

Node.js, C# and .NET have been my bread and butter for quite some time now, with an increased focus on cloud based applications (especially AWS), for quite some time now. Until a few months ago, my day job used to be working on a pretty large, distributed web app that mostly satisfies the guidelines to be called a twelve-factor app.

Outside of my job, I was fortunate in contributing to a few Open Source projects - especially in NodeJS/.NET/iOS/Android/Web etc. Have done a fair bit of Aurelia, Angular & Ember and I used to write web and/or desktop/mobile apps (with tools like PhoneGap/Cordova, nw.js/electron etc.). Off late though, I haven’t been able to find as much time as I would like on these OSS projects.
